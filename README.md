## 🚀 Key Features

### 🧠 Narrative Insight (Powered by OpenCLAW)
Instead of "4h 12m Chrome," Meridian says: *"You were on a roll with that Auth.js refactor, but that 40-minute rabbit hole into mechanical keyboard videos killed your momentum. Stand up, stretch, and get back to the code."*

### 🛑 Distraction Loop Detector
Caught in the Instagram-to-YouTube-to-WhatsApp loop? Meridian detects the pattern and triggers a **5-second Friction Screen**. It doesn't block you—it just forces you to breathe and choose your attention.

### 👻 The Ghost Layer
Privacy isn't an option; it's a right. Toggling Ghost Mode redacts sensitive browser tabs and documents from your AI context. What happens in the Ghost Layer stays in the Ghost Layer.

### 🔋 Cognitive Score
A real-time "Fuel Gauge" for your brain. It calculates focus depth, recovery speed, and task velocity to tell you when to push and when to pivot.

## 🌊 The Meridian Flow

1.  **Capture**: Meridian observes app-switching and focus patterns.
2.  **Analyze**: OpenCLAW processes the telemetry into a "Mental Snapshot."
3.  **Intervene**: If a distraction loop is detected, a 5s Friction Screen triggers.
4.  **Insight**: End of day, you get a Narrative Summary of your cognitive performance.

## 🏃 How to Run the Demo

### Prerequisites
*   Node.js (v18+)
*   OpenCLAW CLI installed and configured.

### Installation
```bash
# Clone the repository
git clone https://github.com/Sriya-Rani-Khadanga/meridian.git
cd meridian

# Install dependencies
npm install

# Start the Cognitive OS
npm run dev
```
Visit `http://localhost:3000` to enter the flow.
